Risk & Return

CO2 · COMPUTE RISK & RETURN FOR SINGLE SECURITY AND PORTFOLIO

1. Return for a Single Security Ch. 10

The expected return of a security is the probability-weighted average of all possible returns. Risk is measured by the variance or standard deviation of returns — how much actual returns deviate from the expected.

Expected Return & Variance
E(R) = Σ pᵢ × Rᵢ
σ² = Σ pᵢ × (Rᵢ – E(R))²
pᵢ = probability of state i  |  Rᵢ = return in state i  |  σ = standard deviation (risk)
Reliance Industries — Scenario Analysis

Boom (p=30%): R = 40% | Normal (p=50%): R = 15% | Recession (p=20%): R = –10%

E(R) = 0.30×40% + 0.50×15% + 0.20×(–10%) = 12% + 7.5% – 2% = 17.5%

σ² = 0.30×(40–17.5)² + 0.50×(15–17.5)² + 0.20×(–10–17.5)² = 151.875 + 3.125 + 151.25 = 306.25

σ = √306.25 = 17.5%

2. Portfolio Return & Risk Ch. 10

A portfolio's expected return is the weighted average of individual returns. However, portfolio risk is not a simple average — it depends critically on the correlation between assets.

Two-Asset Portfolio
E(Rₚ) = w₁E(R₁) + w₂E(R₂)
σ²ₚ = w₁²σ₁² + w₂²σ₂² + 2w₁w₂σ₁σ₂ρ₁₂
w = portfolio weight  |  ρ₁₂ = correlation between assets 1 and 2  (–1 ≤ ρ ≤ +1)

Diversification Magic: When ρ < 1, portfolio risk < weighted average of individual risks. When ρ = –1 (perfectly negatively correlated), risk can be eliminated entirely.

Nifty IT vs Nifty FMCG Portfolio

IT stock: E(R) = 18%, σ = 25%. FMCG stock: E(R) = 12%, σ = 14%. Correlation ρ = 0.20. Equal weights (50/50).

E(Rₚ) = 0.5×18% + 0.5×12% = 15%

σ²ₚ = 0.25×625 + 0.25×196 + 2×0.5×0.5×25×14×0.20 = 156.25 + 49 + 35 = 240.25

σₚ = √240.25 ≈ 15.5% — less than the weighted average of 19.5%!

3. Systematic vs Unsystematic Risk Ch. 11

🌍 Systematic Risk (Market Risk)

Affects ALL stocks — cannot be diversified away. Examples: RBI rate hikes, inflation, global recession, COVID-19. Measured by Beta (β).

🏢 Unsystematic Risk (Firm-specific)

Affects only a specific company or industry. Examples: Byju's accounting scandal, Adani short-seller report. Can be diversified away by holding a broad portfolio.

💡

Key Insight: Rational investors hold diversified portfolios, so the market only compensates them for systematic risk. Unsystematic risk earns no extra return — you can eliminate it for free by diversifying.

4. Capital Asset Pricing Model (CAPM) Ch. 11

CAPM establishes the relationship between a stock's systematic risk (beta) and its required return. Beta measures a stock's sensitivity to market movements.

CAPM — Security Market Line
E(Rᵢ) = Rƒ + βᵢ × (E(Rₘ) – Rƒ)
Rƒ = risk-free rate (10-yr G-Sec yield ≈ 7%)  |  βᵢ = beta of stock i  |  E(Rₘ) – Rƒ = market risk premium (India ≈ 5–7%)
BetaInterpretationIndian Example
β = 0No market sensitivity (risk-free)Government T-Bills
β = 0.5Half as volatile as marketFMCG/utilities (ITC, NTPC)
β = 1.0Moves exactly with marketNifty index fund
β = 1.550% more volatile than marketMidcap IT, PSU banks
β > 2Very aggressiveSmall-cap growth stocks
CAPM Calculation — Infosys

Rƒ = 7% (10-yr G-Sec). Market risk premium = 6%. Infosys Beta = 0.9.

Required return = 7% + 0.9 × 6% = 7% + 5.4% = 12.4%

This 12.4% is used as the required rate of return on equity (Kₑ) in the WACC calculation.
